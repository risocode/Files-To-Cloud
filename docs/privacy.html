Privacy Policy

Last Updated: June 2026

📌 Overview

Files ➪ Cloud is a Telegram-based file storage and management service. Your privacy and data security are important to us.

📂 Information We Collect

  • Telegram User ID
  • Telegram Username
  • Display Name
  • Uploaded Files
  • Folder Information
  • Storage Usage Statistics
  • Favorite and Search Preferences

🔒 How We Use Data

  • Provide file storage services
  • Organize folders and uploads
  • Maintain account statistics
  • Improve service quality
  • Prevent abuse and unauthorized access

☁️ File Storage

Files uploaded through the bot may be stored inside Telegram channels, Telegram cloud infrastructure, or supporting storage systems required for operation of the service.

🚫 Data Sharing

We do not sell, rent, or intentionally share personal data with third parties. Files remain accessible only through authorized access methods provided by the service.

🛡 Security

Reasonable measures are implemented to protect stored files and user information. However, no internet-based service can guarantee absolute security.

🗑 Data Removal

Users may delete files and folders using the available bot functions. Deleted content may be removed permanently according to operational requirements.
