Files ➪ Cloud

🛡 Secure • 🔒 Private • ⚡ Fast • ☁️ Always with you

Files ➪ Cloud is a Telegram-powered cloud storage bot that helps users upload, organize, search, favorite, and retrieve files directly from Telegram.

⌄
🔒 Privacy 📁 Data Usage ☁️ Storage 💬 Contact

✅ Privacy Summary

We only store the information needed to operate the bot: Telegram account identifiers, file metadata, folder structure, favorites, tags, and storage references. We do not sell user data.

🔒 Privacy

Your privacy is treated as a core part of the service. Files ➪ Cloud does not collect passwords, payment information, or private Telegram conversations.

📁 Data Usage

  • Telegram User ID, username, and display name
  • Uploaded file names and metadata
  • Folder names, tags, favorites, and search information
  • Storage references needed to retrieve files

☁️ Storage

Files uploaded through the bot may be archived using private Telegram storage channels and referenced in the bot database for search and retrieval.
