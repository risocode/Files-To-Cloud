🔒 Privacy Policy

We respect user privacy. The bot only stores information needed to provide file storage, organization, search, retrieval, and account-level file management.

We do not sell user data. We do not collect passwords, payment data, or private Telegram conversations.

📂 Information We May Store

  • Telegram User ID
  • Telegram username and display name
  • Uploaded file metadata
  • Original and stored file names
  • Folder names and folder structure
  • File category, size, upload date, tags, and favorites
  • Archive channel message references used for retrieval

☁️ How Files Are Stored

Files uploaded through the bot may be copied to private Telegram archive channels and referenced in the bot database so users can search and download their own files later.

Telegram infrastructure may process and store uploaded media according to Telegram's own policies.

🛠 How Data Is Used

  • To save and organize uploaded files
  • To support search, folders, favorites, and recent files
  • To display storage usage and file counts
  • To prevent abuse and protect user access
  • To troubleshoot service errors and reliability issues

🗑 Data Deletion

Users may delete files using available bot controls. Deleted file records may be removed or marked deleted in the system database depending on service operations.

For full account or data deletion requests, contact the administrator.
